An interactive stock visualization dashboard built with D3.js, showcasing Apple (AAPL) stock performance over time. This project demonstrates dynamic charting, zooming, and brushing techniques for financial data exploration.

## Live demo

https://liv504.github.io/d3-trading-dashboard/

## Features

- **Area Chart Visualization** of AAPL adjusted closing prices  
- **Zoom & Pan**
  - Mouse wheel to zoom
  - Middle mouse drag to pan
  - Touch support  
- **Brush Selection**
  - Select a time range to zoom into specific periods  
- **Quick Time Filters**
  - Last 5 Years
  - Last 1 Year
  - Last 1 Month
  - Last 1 Week  
- **Responsive SVG layout** using `viewBox`
